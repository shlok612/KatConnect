## 📖 Overview

**KatConnect** is a real-time chat web app designed for smooth, instant communication.  
Built using the **MERN** stack and **Socket.io**, it allows users to create accounts, log in securely, and chat with others instantly — all with real-time updates.

---

## ✨ Features

- 🔒 **Authentication System** — Secure login and signup using JWT.  
- 💬 **Real-Time Messaging** — Powered by **Socket.io** for instant updates.  
- 🧑‍🤝‍🧑 **Online Status** — See when users are online.  
- ⚡ **Modern UI** — Built with **React + TailwindCSS**, ensuring responsive and elegant design.  
- 📱 **Fully Responsive** — Works seamlessly on mobile, tablet, and desktop.  
- 🧠 **RESTful APIs** — Clean API architecture for scalability.  
- 🚀 **Deployed on Render** — Both frontend and backend hosted for smooth live performance.

---

## 🛠️ Tech Stack

**Frontend:**
- React.js (Vite)
- Tailwind CSS
- Axios
- Context API / Custom Hooks

**Backend:**
- Node.js
- Express.js
- MongoDB (Mongoose)
- Socket.io
- JSON Web Tokens (JWT)
- bcrypt.js (for password hashing)

**Deployment:**
- Render (for both frontend & backend)
